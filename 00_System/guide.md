# Vault Guide

This is the DECA Euro Windows & Doors production vault — a CRM and knowledge base managed in Obsidian.

## Folder Overview

| Folder | Purpose |
|--------|---------|
| `00_System` | Templates, guides, agent instructions |
| `01_Customers` | One note per customer with contact info, linked deals |
| `02_Sellers` | Team member profiles — Eric, Paul, Ilya |
| `03_Products` | Product specifications — GEALAN LINEAR, S8000 IQ |
| `04_Deals` | One note per deal/opportunity with stage tracking |
| `05_Interactions` | Call logs, email summaries, meeting notes |
| `06_Knowledge` | Sales process, objection handling, company info |
| `07_Analytics` | Reports, win/loss analysis, pipeline insights |
| `08_Inbox` | Quick capture — sort into proper folders later |
| `09_Research` | Archived research from the vault design phase |

## Conventions
- Every note has YAML frontmatter (type, status, created, tags)
- Link related notes with `[[wiki-links]]`
- Use templates from `00_System/Templates/` for consistency
