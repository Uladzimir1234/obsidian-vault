# "Build Your Second Brain With Claude Code & Obsidian" — WhyTryAI

**Published 2 days ago (Feb 26, 2026)** — very current.

## Core Thesis
> "Obsidian isn't for me. It's for Claude Code."

The vault is designed to be the **context layer for AI agents**, not just for human note-taking.

## Why Obsidian is Perfect for AI
1. **Files are local** — agent has filesystem access
2. **Markdown format** — exactly what LLMs prefer
3. **Wiki-links [[]]** — trivial for AI to navigate
4. **Agent-agnostic** — not locked to any specific AI tool

## Key Insight: Agent-Agnostic Context
> "I'm building a standalone, organized context repository, untethered from any specific agent. When a new awesome agent comes along, I'll have something I can point at and say: Here, this is me and everything I'm working with. Let's go!"

This is exactly our situation with OpenClaw → future agents.

## Why PKM Apps Supplement AI
- **Interconnected** — deep connections between objects spanning notes
- **Visual** — graph view, dashboards (AI chats can't do this)
- **Accessible on the go** — mobile apps, sync
- **Agent-agnostic** — knowledge not locked in one AI tool

## Setup: 3 Steps
1. Install Obsidian (free)
2. Create vault as subfolder in agent's working directory
3. Ask the agent to set things up
