# Kyle Gao — "Using Claude Code with Obsidian - A Perfect Pairing"

## Core Insight
> "The tasks I avoid in Obsidian — organization, consistency, following procedures, grunt work — are exactly what AI excels at."

## What AI Does in His Vault
1. **Automated backlinking** — "read my journal and add [[links]] to all people, places, books" → 10-15 min work in seconds
2. **Formatting rough notes** — brain dump → structured review
3. **Mental model application** — systematic decision frameworks
4. **Retrospectives** — comparing plan vs reality with analysis

## Why It Works
- Obsidian vault = essentially a "codebase of markdown files"
- AI already excellent at navigating file structures
- AI doesn't get tired of repetitive organization work
- Natural language interface > learning complex plugins

## Key Quote
> "Tasks that felt like chores now feel effortless. I find myself using Obsidian more frequently because Claude Code removes the friction."

## 💡 Takeaways for DECA
- **AI as the organizer** — we dump data in, AI structures it
- **Auto-backlinking** = huge for us: every customer mention auto-linked to their page
- **Grunt work = AI work** — creating customer files, updating stages, linking products
- **"Friction removal"** principle — if the system is easy, it gets used; if it's work, it doesn't
- Our unified-collector already does the grunt work → just need to output TO vault format
