# Obsidian Agent Skills — Official (by CEO)

## What It Is
Official Agent Skills created by **Kepano, CEO of Obsidian** himself. Teaches AI agents how to properly work with Obsidian files.

## Skills Included
| Skill | What It Teaches AI |
|-------|-------------------|
| **obsidian-markdown** | Create/edit Obsidian-flavored Markdown: wikilinks, embeds, callouts, properties |
| **obsidian-bases** | Create/edit Bases (.base files): views, filters, formulas, summaries |
| **json-canvas** | Create/edit Canvas files: visual boards with nodes, edges, groups |
| **obsidian-cli** | Interact with vaults via CLI |
| **defuddle** | Extract clean markdown from web pages |

## Compatible With
- Claude Code
- Codex CLI
- OpenCode
- Any skills-compatible agent

## Installation
Can be added to `.claude/` folder in vault root, or via npx/marketplace.
